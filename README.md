# 📝 Note App 4

Note App 4 is a simple and efficient Flutter application designed to help users take and manage notes easily. It uses **Flutter**, **Hive** for local storage, and **Bloc/Cubit** for state management.

## 📱 Features

- ✅ Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 💾 Store notes locally using Hive
- 🎯 Clean and responsive UI
- ⚡ Fast performance and offline access
- 📦 Bloc pattern for organized state management
